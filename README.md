# Blog Platform with Comments

## Project Overview
The Blog Platform with Comments is a full-stack web application that allows users to create, edit, and manage blog posts. Users can also comment on blogs and interact with other users through a simple and responsive interface.

---

# Features

- User Authentication & Authorization
- Create, Edit, and Delete Blog Posts
- Comment System for Blogs
- User-Friendly Interface
- Responsive Design for Mobile and Desktop
- Secure Login & Registration
- REST API Integration

# Installation

## Clone Repository

```bash
git clone https://github.com/your-username/blog-platform.git
```

## Install Dependencies

### Frontend
```bash
cd client
npm install
```

### Backend
```bash
cd server
npm install
```

---

# Run the Application

## Start Backend Server
```bash
npm start
```

## Start Frontend
```bash
npm run dev
```
